# 清华大学研究生学位论文格式规范

> 来源：《清华大学研究生学位论文写作指南（202605）》及 thuthesis-v7.7.1

## 1. 页面设置

| 项目 | 规范值 |
|------|--------|
| 纸张 | A4 (21.0cm × 29.7cm) |
| 页边距 | 上 3.0cm，下 3.0cm，左 3.0cm，右 3.0cm |
| 装订线 | 0cm |
| 页眉距边界 | 2.2cm |
| 页脚距边界 | 2.2cm |

## 2. 篇眉与页码

- 篇眉从"摘要"开始，内容与该部分章标题相同
- 篇眉：五号字(10.5pt)居中，中文宋体，英文 Times New Roman
- 页码从"摘要"开始，摘要至符号说明用罗马数字，正文起用阿拉伯数字
- 页码：Times New Roman 五号字，居中，两侧不加修饰线

## 3. 标题格式

### 章标题（Heading 1）
- 命名格式："第1章　引言"（章序号与标题间空一个汉字符，即全角空格 \u3000）
- 字号：三号(16pt)
- 字体：中文黑体，英文和数字 Arial
- 对齐：居中
- 行距：单倍行距
- 段前：24磅（480 twips）
- 段后：18磅（360 twips）
- 每章另起页（pageBreakBefore）

### 一级节标题（Heading 2）
- 命名格式："2.1 实验装置"（节序号与标题间空一个汉字符）
- 字号：四号(14pt)
- 字体：中文黑体，英文和数字 Arial
- 对齐：居左
- 行距：固定值 20磅（400 twips，lineRule=exact）
- 段前：24磅（480 twips）
- 段后：6磅（120 twips）

### 二级节标题（Heading 3）
- 字号：13pt
- 字体：中文黑体，英文和数字 Arial
- 对齐：居左
- 行距：固定值 20磅
- 段前：12磅（240 twips）
- 段后：6磅（120 twips）

## 4. 正文段落

- 字号：小四号(12pt)
- 字体：中文宋体，英文 Times New Roman
- 对齐：两端对齐(JUSTIFY)
- 首行缩进：2个汉字符（约24pt = 480 twips）
- 行距：固定值 20磅（400 twips，lineRule=exact）
- 段前：0磅
- 段后：0磅

## 5. 图

- 图序与图题置于图的下方
- 编号：按章编号，如"图2.1"（前一位为章号，后一位为本章内顺序号）
- 图序与图题间空一个汉字符（全角空格 \u3000）
- 字号：11pt
- 字体：中文宋体，英文和数字 Times New Roman
- 对齐：居中
- 行距：单倍行距
- 段前：6磅（120 twips）
- 段后：12磅（240 twips）
- 不加粗、不斜体
- 不需要单独的来源行（引用来源在正文中用 [1] 标注）

## 6. 表

- 采用三线表
- 表序与表题置于表的上方
- 编号：按章编号，如"表3.1"
- 表序与表题间空一个汉字符
- 字号：11pt
- 字体：中文宋体，英文和数字 Times New Roman
- 对齐：居中
- 行距：单倍行距
- 段前：12磅（240 twips）
- 段后：6磅（120 twips）

### 三线表边框规范
- 上边线（顶线）：单直线，线粗 1.5磅（sz=12，单位为八分之一磅）
- 表头下边线（第三条线）：单直线，线粗 1磅（sz=8）
- 下边线（底线）：单直线，线粗 1.5磅（sz=12）
- 左右边线：无（nil）
- 中间行：无横线

### 表单元格文字
- 字号：11pt
- 字体：中文宋体，英文和数字 Times New Roman
- 对齐：居中（上下居中，左右居中）
- 行距：单倍行距
- 段前：3磅
- 段后：3磅
- 表头行加粗

## 7. 参考文献

- "参考文献"标题格式与章标题相同
- 字号：五号(10.5pt)
- 字体：中文宋体，英文和数字 Times New Roman
- 行距：固定值 16磅（320 twips，lineRule=exact）
- 段前：3磅（60 twips）
- 段后：0磅
- 格式：悬挂缩进 2个汉字符（约21pt = 420 twips）
  - left_indent = 420 twips
  - first_line_indent = -420 twips

## 8. 封面

- 论文题目：一号黑体(26pt)，居中
- 副标题/类型：三号宋体(16pt)，居中
- 信息：小四号(12pt)宋体，居中

## 9. 目录

- 标题"目录"：黑体三号字，居中，单倍行距，段前24磅段后18磅
- 目录中章标题：小四号黑体，居左
- 目录中节标题：小四号宋体，缩进1-2个汉字符
- 行距：固定值20磅
- 使用 Word TOC 域自动生成

## 10. 字体颜色

- 全文所有字体颜色统一为黑色(#000000)
- 需同时修改样式定义和 run 级别颜色
- Word 内置 Heading 样式默认有蓝色(如 365F91, 4F81BD)，必须覆盖

## 11. python-docx 单位换算

| 单位 | 换算 |
|------|------|
| 1磅(pt) | 12700 EMU |
| 1磅(pt) | 20 twips |
| 1cm | 360000 EMU |
| 1汉字符 | 约12pt = 240 twips |
| 字号sz属性 | 半磅，如16pt → sz=32 |
| 边框sz属性 | 八分之一磅，如1.5磅 → sz=12 |
